# 🔍 ГИРБО Mining Parser

Инструмент для поиска майнинговых компаний через бухгалтерскую отчётность (ГИРБО/ФНС).

Скрипт автоматически:
- Собирает ИНН компаний по ОКВЭД и регионам из ЕГРЮЛ
- Загружает бухотчётность из ГИРБО
- Ищет расходы на электроэнергию в пояснениях
- Считает скоринг "похожести на майнинг"
- Выдаёт приоритизированный CSV со списком лидов

## ⚡ Быстрый старт

### 1. Клонировать репозиторий
```bash
git clone https://github.com/your-username/girbo-mining-parser.git
cd girbo-mining-parser
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Настроить параметры
Отредактировать `config.py` — выбрать регионы, ОКВЭД, пороги.

### 4. Запустить

**Автоматический режим** (ИНН тянутся из ЕГРЮЛ):
```bash
python main.py --mode api
```

**Из файла** (быстрее, если есть выгрузка из СПАРК/Контур):
```bash
python main.py --mode file --input data/inns.txt
```

**С кастомным порогом ЭЭ:**
```bash
python main.py --mode file --input data/inns.txt --min-ee 10000000
```

## 📊 Формат результата

Файл `output/mining_leads_YYYY-MM-DD.csv`:

| Колонка | Описание |
|---|---|
| ИНН | ИНН компании |
| Компания | Краткое наименование |
| Регион | Регион регистрации |
| ОКВЭД | Основной ОКВЭД |
| Расходы_ЭЭ_руб | Расходы на электроэнергию (руб.) |
| Выручка_руб | Годовая выручка |
| Себестоимость_руб | Себестоимость |
| ОС_руб | Основные средства |
| Баланс_руб | Валюта баланса |
| Скоринг_майнинг | Балл 0–100 |
| Приоритет | 🔴 Горячий / 🟡 Тёплый / 🟢 Холодный |

## ⚙️ Скоринговая модель

| Признак | Баллы |
|---|---|
| Расходы на ЭЭ > 50 млн руб. | +30 |
| Расходы на ЭЭ 10–50 млн руб. | +20 |
| Себестоимость / Выручка > 70% | +15 |
| ОС / Баланс > 50% | +20 |
| Мало сотрудников при высокой выручке | +15 |
| Любые расходы на ЭЭ найдены | +20 |

---

## ⚠️ Ограничения

- ГИРБО раскрывает расходы на ЭЭ только если компания их указала в пояснениях
- Малые предприятия (выручка < 800 млн) могут сдавать упрощённую отчётность без пояснений
- API ЕГРЮЛ имеет rate-limit — не убирать `REQUEST_DELAY`
- Данные обновляются в ГИРБО раз в год (после сдачи отчётности)
